
# Demo Script (5 minutes)

**Goal:** Show how product usage translates to business value for a customer.

1) **Context (30s):** “I built a lightweight ROI dashboard to help quantify value during discovery.”
2) **Login (15s):** Use demo credentials.
3) **Usage (60s):** “Here’s a 90‑day adoption curve generated from sample data.”
4) **Assumptions (60s):** Adjust license and implementation costs; watch ROI update.
5) **Value (60s):** Walk through Total Savings, Net Benefit, Payback.
6) **Collateral (45s):** Click “Download PDF Report” to generate exec‑ready artifact.
7) **Next steps (30s):** “We can connect HubSpot or CSV exports to replace mock data.”
